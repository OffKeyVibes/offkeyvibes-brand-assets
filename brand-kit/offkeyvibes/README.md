# OffKeyVibes Brand Kit

This folder turns the current OffKeyVibes visual world into a reusable system for future posts, mockups, prompts, and quick graphics.

Everything here is based on assets that are already public or already in the brand-assets repo:
- OffKeyVibes horizontal logo
- OKV mark
- core desktop wallpapers
- wide band banner
- song-era art such as `Break The Chains`

## What is here

- `tokens/`
  - exact palette tokens in JSON and CSS form
- `docs/`
  - style guide, identity map, and copy bank
- `prompts/`
  - reusable prompt recipes for future AI image generation
- `generated/social/`
  - ready-made post graphics and quick memory assets
- `tools/`
  - local Pillow builder for regenerating the included graphics

## Why this exists

OffKeyVibes is still early, so the brand system needs to be honest and reusable:
- heavy and memorable
- clear about `okv.rocks`
- usable before a full release catalog exists
- consistent across posts, covers, previews, and future generated art
- centered on the main OffKeyVibes identity first, not any single song era

## Current guidance

Use `okv.rocks` as the memory anchor in quick-share graphics.

Use `OffKeyVibes` as the identity anchor on homepage, posters, and band-first pieces.

Use the core ember/red/orange identity as the default visual lane.

Use `Break The Chains` as one song-era example, not the full basis of the brand.

Treat blue looks as experiments unless a future era intentionally adopts them.
