# OffKeyVibes Image Prompt Recipes

These are future-ready prompt specs for OpenAI image generation. They are written to stay honest to the current public state of the project and to keep the core OffKeyVibes brand separate from any one song era.

## 1. OffKeyVibes Core Identity Wallpaper

Use case: stylized-concept
Asset type: wallpaper or wide identity background
Primary request: create a clean, powerful OffKeyVibes core-brand wallpaper that feels like the default visual identity of the artist
Subject: the OffKeyVibes logo and ember-field atmosphere
Style/medium: cinematic wallpaper, dark editorial band identity, not busy
Composition/framing: centered logo, strong negative space, atmospheric background texture, no clutter
Lighting/mood: ember glow, burn haze, pressure, restrained sparks
Color palette: black, burnt rust, signal orange, molten gold, bone white
Constraints: should feel usable for desktop wallpaper, headers, and brand identity posts
Avoid: fake crowd shots, busy collages, purple neon, cyberpunk blue, cheesy chrome, song-title text, oversharpening

## 2. okv.rocks Memory Graphic

Use case: stylized-concept
Asset type: social square reminder graphic
Primary request: create a heavy, high-impact social graphic that helps people remember the domain okv.rocks
Subject: the OffKeyVibes identity and the exact text `okv.rocks`
Style/medium: dark editorial band poster, not glossy, not comic, not neon
Composition/framing: centered composition, strong logo area, huge readable domain, space for one small supporting line
Lighting/mood: ember glow, burn haze, pressure, restrained sparks
Color palette: black, burnt rust, signal orange, molten gold, bone white
Text (verbatim): `okv.rocks`
Constraints: readable at phone size, no fake streaming platform logos, no fake release claims
Avoid: purple neon, cyberpunk blue, cheesy chrome, random skull clutter, fake crowd shots, oversharpening

## 4. Homepage Hero Background Pack

Use case: stylized-concept
Asset type: background image for homepage or teaser page
Primary request: create a cinematic heavy-music background for OffKeyVibes that supports overlaid text without getting busy
Subject: abstract pressure, fire haze, chain tension, shadowed movement
Style/medium: cinematic concept art background, no visible text, no logos baked into the image
Composition/framing: central breathing room for headline and CTA, detail pushed toward edges
Lighting/mood: dark and smoldering, volatile but controlled

## 6. Song-Era Status Graphic

Use case: stylized-concept
Asset type: social update tile
Primary request: create a heavy status graphic for one OffKeyVibes song era such as Break The Chains without turning that era into the whole band identity
Subject: one title, one status line, one anchor URL
Style/medium: premium metalcore-adjacent promo graphic, restrained and real
Composition/framing: one title, one status line, one anchor URL, clear separation from core brand posts

Constraints: do not imply the song is already out, do not add streaming badges
Avoid: fake release dates, fake streaming claims, visual choices that replace the core OffKeyVibes identity
