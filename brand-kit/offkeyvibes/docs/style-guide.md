# OffKeyVibes Style Guide

## Core Read

OffKeyVibes should feel like pressure turning into release.

The mood is not polished pop aggression. It is heavier, more personal, and more scorched:
- anger with purpose
- emotional pressure
- survival energy
- heat, rust, smoke, chains, sparks, impact

The brand should feel like something built in the middle of stress, not in a clean studio vacuum.

## Brand Position

Use this as the working public read:

`OffKeyVibes is an original heavy music project out of Cassatt, South Carolina.`

Good follow-up descriptors:
- heavy original music
- pressure, rage, release
- built from stress and survival
- building in public

## Identity Hierarchy

OffKeyVibes has a core identity and then separate song eras inside it.

Default order of truth:
1. OffKeyVibes brand identity
2. `OKV` shorthand / memory mark
3. `okv.rocks` domain recall
4. song-era assets like `Break The Chains`

Do not let one single become the whole brand unless that is an intentional long-term decision later.

## Visual DNA

The current visual system is built from:
- blackened ember backgrounds
- rust and burnt copper accents
- molten orange highlights
- gold warmth in the logo
- bone-white logo text for contrast
- subtle deep-violet shadow tones in the mark

The strongest current core references are:
- the main OffKeyVibes logo on transparent background
- the fiery OffKeyVibes desktop wallpaper
- the `OKV` desktop wallpaper
- the `okv.rocks` desktop wallpaper

Textures and motions that fit:
- smoke
- burn haze
- crackling light
- lightning fractures
- worn metal
- chain tension
- ash and grit

## Song-Era Rule

Song art can push a slightly different mood, but it should still read as belonging to OffKeyVibes.

`Break The Chains` is a valid song-era template:
- chains
- restraint
- impact
- body-under-pressure imagery

But it is not the entire basis of the brand.

## Color Rule

Primary lane:
- obsidian black
- ember red
- rust / burnt copper
- signal orange
- molten gold
- bone-white
- deep violet shadow inside the logo

Blue lane:
- blue has appeared in generated experiments
- blue is not the default OffKeyVibes identity right now
- only use blue on purpose for an alternate concept or future song era
- do not let a cool blue image rewrite the main brand system by accident

## Composition Rules

Keep the page or image structure simple and forceful:
- one main thing to remember
- one current status line
- one next action

Good hierarchy:
1. `OffKeyVibes` or `okv.rocks`
2. current message
3. supporting line

Avoid filling empty space just to make a design feel busier.

Wallpaper-style branding works well for OffKeyVibes:
- centered mark
- strong ember field
- clean negative space
- simple memory message

That direction is more honest and durable than over-explaining every image.

## Homepage Rule

The homepage should explain the project in order:
1. who OffKeyVibes is
2. what is happening now
3. where to go next

That pipeline is stronger than a generic teaser poster.

## Tone Rule

Say what is true.

When there is little public material, the site and posts should still feel strong, but they should not pretend a full release slate already exists.

Good:
- `Break The Chains is on the way.`
- `Lyrics are live now.`
- `Track release still in progress.`
- `Start at okv.rocks.`

Avoid:
- fake countdown energy
- fake release-final language
- overpromising phrases like `out everywhere now` unless it is actually true

## Location Usage

Recommended long form:
- `Out of Cassatt, South Carolina`

Recommended neutral form:
- `Cassatt, South Carolina, United States`

Use the location sparingly. It should ground the project, not become the whole message.

## What To Avoid

Avoid these directions unless a future era explicitly changes the brand:
- bright candy colors
- soft pastel gradients
- clean corporate minimalism
- chrome sci-fi blue
- purple-heavy fantasy looks
- pop-punk collage chaos
- cute horror kitsch
- fake crowd-shot festival branding

## Repeatable Brand Words

Useful recurring words:
- pressure
- weight
- rage
- release
- fight
- burn
- survive
- rise
- chains
- strain
- scarred
- heavy
- within
- pit
- lights out

Use them with restraint. The goal is identity, not parody.
