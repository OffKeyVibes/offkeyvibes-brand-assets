- `debut album dropping soon` unless true
- `stream now` unless true
- `official release out everywhere` unless true
- `worldwide sensation` style hype

## Image Prompt Notes

When prompting future images, keep these constants:
- heavy but honest
- dark ember palette
- industrial burn energy
- no polished glam-metal cliches
- no purple neon default
- no blue as the default lane
- no fake concert crowd unless it represents a real live image
