## Short-Memory System

Use these three in combination:

- `OffKeyVibes` for identity
- `OKV` for shorthand
- `okv.rocks` for memory and routing

This is the easiest mental pipeline for people:

1. See OffKeyVibes
2. Remember OKV
3. Type okv.rocks

## Color Policy

Main palette:
- ember reds
- scorched black
- molten orange
- gold warmth
- logo white / silver
- deep violet shadow

Experimental palette:
- blue can exist as an alternate concept
- blue is not the default brand lane
- blue should stay opt-in and clearly intentional

## Future Use Rule

When you create something new, decide which lane it belongs to before building it:

- core identity
- domain memory
- wallpaper
- homepage / site
- song era
- experiment

That one choice will keep the repo cleaner and prevent visual drift.
